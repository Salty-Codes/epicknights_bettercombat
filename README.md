# Epic Knights x BetterCombat

_A datapack reworking the BetterCombat integration for Epic Knights_

## What does it do?

**Epic Knights** integration of **Better Combat** is good. But most weapon types share the same pattern, which makes combat feel a bit repetitive regardless of what you're wielding.

This datapack tries to fix that by giving every weapon type a **unique combo sequence** built around how that weapon is somehow actually used.
